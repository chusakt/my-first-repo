# Documentation Directory

This folder contains all project documentation files.

## Files

- **README.md** - Project overview and quick start guide
- **REQUIREMENTS.md** - Software Requirements Specification (SRS)
- **ARCHITECTURE.md** - System architecture and design
- **USER_GUIDE.md** - User manual and tutorials
- **CHANGELOG.md** - Version history and roadmap
- **TECHNICAL_SPECS.md** - Technical specifications
- **DEVELOPMENT.md** - Developer guide and best practices
- **DOCUMENTATION.md** - Documentation overview

## Documentation Hierarchy

For users → Start with README.md then USER_GUIDE.md

For developers → Start with README.md then DEVELOPMENT.md

For project managers → Start with README.md then REQUIREMENTS.md

For architects → Start with ARCHITECTURE.md then TECHNICAL_SPECS.md

**Note**: After moving documentation files here, update links in README.md if necessary.
